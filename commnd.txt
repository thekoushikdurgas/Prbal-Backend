# Django settings

# Database (Supabase PostgreSQL)

## 5. Setup Commands

Here are all the commands you need to set up and run your WanderAI project:

### Backend Setup

```bash
# 1. Navigate to backend directory
cd wanderai-backend

# 2. Create a Python virtual environment
python -m venv venv

# 3. Activate the virtual environment
# On Windows
venv\Scripts\activate
# On macOS/Linux
source venv/bin/activate

# 4. Install dependencies
pip install -r wanderai/requirements/development.txt

# 5. Load environment variables
# Install python-dotenv globally if not installed
pip install python-dotenv

# 6. Run migrations
cd wanderai
python manage.py migrate

# 7. Create superuser
python manage.py createsuperuser

# 8. Run the development server
python manage.py runserver
```

### Flutter/Dart Frontend Setup

```bash
# 1. Navigate to frontend directory
cd wanderlustai

# 2. Install dependencies
flutter pub get

# 3. Run the app in development mode
flutter run
```

### Deployment to Fly.io

```bash
# 1. Install flyctl if not installed
# On Windows with PowerShell
iwr https://fly.io/install.ps1 -useb | iex
# On macOS/Linux
curl -L https://fly.io/install.sh | sh

# 2. Login to Fly.io
fly auth login

# 3. Launch the app (first deployment)
fly launch

# 4. Deploy updates
fly deploy
```

## 6. Setup API Keys and Secrets

### Getting OpenRouter API Key

1. Visit [OpenRouter.ai](https://openrouter.ai/)
2. Create an account or sign in
3. Navigate to the Dashboard
4. Create a new API key
5. Copy the key to your `.env` file

### Setting Up Supabase

1. Visit [Supabase.com](https://supabase.com/) and sign up or login
2. Create a new project
3. Go to Project Settings → Database
4. Copy the connection details to your `.env` file
5. Go to Project Settings → API
6. Copy the URL and anon key to your `.env` file

### Setting Up Ola Maps API (Alternative: Use MapBox)

1. Visit [Ola Maps](https://maps.olacabs.com/)
2. Register for an API key
3. Copy the API key to your `.env` file

If Ola Maps API is unavailable, you can use MapBox as an alternative:

1. Visit [MapBox](https://www.mapbox.com/) and create an account
2. Navigate to Access tokens and create a new token
3. Update your code to use MapBox APIs instead

## 8. Docker Setup (Optional)

Create a `docker-compose.yml` file in the root directory:

```yaml
version: '3'

services:
  backend:
    build: ./wanderai-backend
    command: python wanderai/manage.py runserver 0.0.0.0:8000
    volumes:
      - ./wanderai-backend:/app
    ports:
      - "8000:8000"
    environment:
      - DJANGO_SETTINGS_MODULE=wanderai.config.settings.development
    env_file:
      - ./wanderai-backend/.env

  frontend:
    build: ./wanderlustai
    command: flutter run -d web-server --web-port 3000 --web-hostname 0.0.0.0
    volumes:
      - ./wanderlustai:/app
    ports:
      - "3000:3000"
    depends_on:
      - backend
    env_file:
      - ./wanderlustai/.env
```

Create a `Dockerfile` in the `wanderai-backend` directory:

```dockerfile
FROM python:3.10-slim

WORKDIR /app

ENV PYTHONDONTWRITEBYTECODE 1
ENV PYTHONUNBUFFERED 1

RUN apt-get update && apt-get install -y --no-install-recommends \
    build-essential \
    libpq-dev \
    && apt-get clean \
    && rm -rf /var/lib/apt/lists/*

COPY wanderai/requirements /app/requirements
RUN pip install --no-cache-dir -r /app/requirements/development.txt

COPY . .

EXPOSE 8000
```

Run with:

```bash
docker-compose up
```
